# 宝宝日记 隐私政策

更新日期：2026-07-10
生效日期：2026-07-10
版本号：v2.0

## 引言

**宝宝日记**（以下简称"本应用"，Android 包名：`com.babydiary.app`）由个人开发者（以下简称"我们"或"运营者"）运营。本应用是一款宝宝成长记录工具，帮助家长记录宝宝的喂养、睡眠、疫苗、成长等信息。

我们深知个人信息对您和您家庭的重要性，承诺严格按照《中华人民共和国个人信息保护法》《中华人民共和国数据安全法》《中华人民共和国网络安全法》《儿童个人信息网络保护规定》《App 违法违规收集使用个人信息行为认定方法》等法律法规要求，保护您和您宝宝的信息。

**请您在使用本应用前，仔细阅读并充分理解本政策的全部内容。一旦您点击"同意并继续"，即视为您已同意本政策全部条款。**

## 一、我们收集的信息

### 1.1 您主动提供的信息

| 信息类型 | 具体内容 | 用途 |
|---|---|---|
| 账号信息 | 用户名、昵称、密码（加密存储） | 注册登录、账号管理 |
| 宝宝信息 | 姓名、性别、出生日期、头像 | 记录主体标识 |
| 记录数据 | 喂养、睡眠、尿布、体温、成长、疫苗等宝宝日常记录 | 应用核心功能 |
| 家庭协作 | 家庭成员角色、邀请关系 | 多家长协作记录 |

### 1.2 我们自动收集的信息

| 信息类型 | 具体内容 | 用途 |
|---|---|---|
| 设备标识 | 设备唯一 ID（应用首次启动时由 GUID 算法本地生成，非系统设备识别码） | 数据同步冲突归因 |
| 日志信息 | 应用崩溃日志、异常日志（已自动脱敏 Token / 手机号等敏感字段） | 问题排查与应用优化 |

### 1.3 我们 **不收集** 的信息

为最大限度保护您和宝宝的隐私，本应用明确声明**不收集**以下信息：

- 系统设备识别码（IMEI / Android ID / OAID / MAC 地址）
- 精确或粗略位置信息（GPS / 基站 / Wi-Fi 列表）
- 通讯录、短信、通话记录
- 摄像头预览、麦克风录音
- 已安装应用列表
- 浏览历史、剪贴板内容
- 广告标识符（IDFA / GAID）
- 用户行为画像

## 二、信息的使用

收集的信息仅用于以下目的：

1. **提供核心功能**：宝宝记录的创建、查看、统计、分析
2. **数据同步**：在您**主动启用**同步功能时，将本地数据上传至您配置的服务器
3. **家庭协作**：邀请家人共同记录宝宝成长
4. **智能分析**（可选功能）：基于您的记录数据生成 AI 分析报告，**需您主动配置第三方 LLM 服务后方可启用**
5. **问题排查**：通过脱敏日志定位应用异常
6. **服务改进**：基于应用整体使用情况（非个体画像）优化功能

## 三、信息的存储与保护

### 3.1 存储位置

- **本地存储**：所有数据默认存储在设备本地的 SQLite 数据库中，**不开启同步时不会上传任何业务数据到任何服务器**
- **服务端存储**：仅在您**主动启用数据同步**且配置了同步服务器后，数据才会传输到您指定的服务器

### 3.2 存储期限

- 账号数据：保留至您主动删除账号
- 宝宝记录数据：保留至您主动删除
- 日志文件：本地保留 7 天后自动滚动清理
- 同步服务器上的数据：由您配置的服务器管理，本应用不控制其存储期限

### 3.3 安全措施

- 密码使用 **PBKDF2-SHA256（600,000 次迭代）** 加密存储，明文不可逆
- 同步传输强制使用 **HTTPS** 加密
- 日志自动脱敏 Token / 手机号等敏感字段
- 数据库操作前自动本地备份

## 四、信息的共享、转让、公开披露

### 4.1 共享

我们**不会**将您的信息出售或共享给第三方，但以下情况除外：

1. **经您明确同意**：在共享前再次征求您的同意
2. **法律法规要求**：根据法律法规规定、诉讼、政府主管部门要求
3. **您主动启用的第三方服务**：
   - 若您启用 AI 智能分析功能，您主动提交的分析数据将传输至您配置的 LLM 服务（如 OpenAI、Anthropic、智谱 AI 等），受该服务自身的隐私政策约束
   - 该传输行为由您主动触发，我们不对第三方服务的处理行为负责

### 4.2 转让

我们不会将您的个人信息转让给任何第三方。如未来发生合并、收购等资产转让，我们将要求接收方继续受本政策约束，否则要求其重新征求您的同意。

### 4.3 公开披露

我们仅在以下情况公开披露您的信息：

1. 获得您的明确同意
2. 法律法规要求
3. 法院、行政机关命令

## 五、您的权利

根据《个人信息保护法》您享有以下权利：

| 权利 | 行使方式 |
|---|---|
| **访问与复制** | 在应用内随时查看所有数据；可通过"数据同步"导出到您的服务器 |
| **更正** | 在应用内修改宝宝信息、记录内容 |
| **删除** | 在应用内删除单条记录、删除宝宝、注销账号（同时删除关联数据） |
| **撤回同意** | 关闭同步即停止数据上传；注销账号即撤回全部同意 |
| **同步控制** | 随时开启 / 关闭数据同步，本地数据不受影响 |
| **账号注销** | 通过应用内设置或联系开发者邮箱注销账号 |

**注销账号的影响**：注销后所有本地数据、同步数据、账号信息将被删除且**不可恢复**。

## 六、儿童个人信息保护

本应用用于记录宝宝信息，**账号注册人必须为成年人**（监护人）。针对儿童个人信息，我们额外承诺：

1. **不主动收集儿童本人可识别信息**：宝宝信息由监护人代为录入，不要求儿童本人注册或提供信息
2. **不进行儿童画像分析**：不基于儿童信息做商业推荐或画像
3. **不向儿童推送商业信息**：本应用不集成任何广告 SDK
4. **监护人知情同意**：本应用所有儿童信息均由监护人主动录入，视为监护人已知情同意
5. **删除权**：监护人可随时删除宝宝信息或注销账号，删除后数据不可恢复

如监护人发现未经其同意的儿童信息被录入，可联系 `brianpan95@qq.com` 要求删除。
