# 4. Core Product Rules

## Record First

记录永远是第一入口。

## AI Assist

AI 提供整理和理解，不替代用户表达。

## Memory Over Data

成长记录应该形成记忆，而不是数据库。

## Consistent Experience

不同平台保持统一核心体验。

# 6. Maintenance Rules

- 是否符合产品理念
- 是否复用已有组件
- 是否使用 Design Token
- 是否支持多端一致体验
