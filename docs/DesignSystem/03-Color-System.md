# Color System

## Overview

ChildNotes colors should express warmth, trust and calmness.

The system is divided into:

- Brand colors
- Semantic colors
- Surface colors
- Text colors

## Brand Direction

Primary colors should feel:

- Soft
- Natural
- Comfortable

Avoid aggressive saturation because memories are the focus, not visual stimulation.

## Semantic Colors

### Success

Used for completed records, saved memories and positive states.

### Warning

Used carefully for attention-required situations.

### Error

Used only when users need correction or protection.

## Surface Philosophy

Large areas should use gentle neutral backgrounds to allow photos and memory content to stand out.

## Future Expansion

The color system will support:

- Light theme
- Dark theme
- Accessibility contrast rules
- Platform-specific adaptation
